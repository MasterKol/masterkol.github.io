This is a simulation of multiple linked pendulums which form a chain or rope (depending on the value of n). These equations are generated from a generalized equation governing their motion. The system generated requires solving a N x N set of equations, the solving time for which is proportional to O(n3). This makes solving the system difficult for larger systems approaching 40 to 50 links. Two pendulums that are setup in the same position but slightly offset quickly diverge from one another despite their difference being small, this is the "butterfly effect" in action (properly known as sensitivity to initial conditions).
